# Meow Auto Clicker 3.0

[![License: GPL-3.0](https://img.shields.io/badge/License-GPL%203.0-blue.svg)](https://www.gnu.org/licenses/gpl-3.0)

[English](./README.md)  

一个用 Rust 编写的高级鼠标自动点击器，具有精美的 GUI 界面。非常适合自动化任务、游戏和重复性点击操作。

## 🚀 功能特性

### 🖱️ 鼠标控制
- **多种点击类型**: 支持单机和双击操作
- **左/右键点击**: 可以选择左键或右键点击
- **自定义坐标**: 智能模式可以录制和回放精确的鼠标移动序列

### ⚡ 点击模式
- **普通模式**: 固定间隔的简单自动点击
- **智能模式**: 录制和回放复杂的鼠标移动序列
  - 录制鼠标位置和点击动作
  - 保存和加载录制会话
  - 自定义动作之间的间隔时间

### 🎯 热键支持
- **可自定义热键**: 设置您喜欢的组合键用于：
  - 开始/停止点击操作
  - 录制鼠标移动（仅智能模式）
- **智能防抖**: 防止意外多次触发

### ⚙️ 高级设置
- **点击次数控制**: 设置要执行的精确点击次数
- **时间间隔配置**: 调整点击之间的延迟（毫秒）
- **音效反馈**: 启用/禁用录制动作的音频反馈
- **持久化**: 自动保存设置和录制内容

### 📊 实时状态
- **视觉指示器**: 颜色编码状态显示点击、录制或停止状态
- **录制计数器**: 跟踪智能模式下的录制动作数量

## 🛠️ 安装

### 先决条件
- 系统上已安装 Rust 和 Cargo

### 从源码构建
```bash
# 克隆仓库
git clone git@github.com:jcglqmoyx/clicker.git
cd clicker

# 构建项目
cargo build --release

# 运行应用程序
cargo run --release
```

构建后的二进制文件将在 `target/release/clicker` 中可用

## 📖 使用方法

1. **启动应用程序**: 运行可执行文件打开 GUI
2. **配置设置**:
   - 选择鼠标按钮（左键/右键）
   - 选择点击类型（单击/双击）
   - 设置点击次数和时间间隔
3. **设置热键**: 配置您喜欢的组合键
4. **选择模式**:
   - **普通模式**: 用于简单的重复点击
   - **智能模式**: 用于复杂序列（需要先录制移动）
5. **开始点击**: 使用热键开始/停止点击过程

### 智能模式工作流程
1. 启用"智能点击"复选框
2. 将鼠标定位到要录制第一个点击的位置
3. 按下录制热键捕捉位置和点击动作
4. 为序列中的每个步骤重复此操作
5. 使用开始热键回放整个序列

## 🏗️ 技术细节

### 技术栈
- **Rust**: 系统编程语言，提供性能和安全保障
- **FLTK**: 轻量级 GUI 工具包，跨平台兼容
- **DeviceQuery**: 键盘和鼠标输入处理
- **Enigo**: 跨平台输入自动化
- **Rusqlite**: SQLite 数据库用于持久化存储
- **Rodio**: 音频播放用于音效

### 架构
应用程序采用模块化架构，关注点分离清晰：
- **GUI 层**: 基于 FLTK 的界面组件
- **业务逻辑**: 全局状态管理和点击操作
- **持久化**: SQLite 数据库存储设置和录制内容
- **输入处理**: 设备查询和 Enigo 用于输入自动化

## 🔧 配置

设置会自动保存到应用程序数据目录的本地 SQLite 数据库中。您可以：
- 清除录制的序列
- 保存/加载特定的录制会话
- 自定义所有操作参数
